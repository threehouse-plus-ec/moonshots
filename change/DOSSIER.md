## Systems-Engineering Signature

- Strict four-phase architecture: orbit → land → sample return → research station
- Each mission builds explicitly on predecessor capability
- Closed-loop learning between missions (Chang'e 3 backup → Chang'e 4 redesign)
- Strong state coordination via CNSA
- Queqiao relay infrastructure as prerequisite for far-side operations
- Progressive expansion toward crewed landing (~2030) and ILRS
